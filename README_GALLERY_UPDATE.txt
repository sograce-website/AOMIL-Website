AOMIL V2.5 Gallery Product Pages Update

This package updates product detail pages so they can display:
- full product image gallery
- product videos when available
- product specification table
- inquiry call-to-action

Important:
This package expects the full media patch to be uploaded to:
assets/products/<slug>/media-XX.jpg
assets/products/<slug>/video-XX.mp4

Upload this after or together with the media patch packages.
